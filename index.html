What is the difference between var, let, and const?

Use let for variables that need to change.

Use const for variables that shouldn't change.

Avoid var in modern JavaScript.

What is the difference between map(), forEach(), and filter()

Use map() when you want to create a new transformed array.

Use forEach() when you just want to iterate without returning a new array.

Use filter() when you need to create a subset of an array based on a condition.
